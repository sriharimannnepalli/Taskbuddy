# Task Buddy

A simple and efficient task management tool to help you stay organized and productive.


## About

Task Buddy is designed to be a user-friendly task management application. It aims to provide a clean and intuitive interface for creating, managing, and tracking your tasks. Whether you're working on personal projects or collaborating with a team, Task Buddy can help you streamline your workflow.

## Features

*   **Task Creation:** Easily add new tasks with titles, descriptions, due dates, and priorities.
*   **Task Lists:** Organize tasks into lists or projects for better categorization.
*   **Task Deletion:** Remove completed or unnecessary tasks.
*   **Status Tracking:** Monitor the progress of your tasks with status indicators (e.g., To Do, In Progress, Completed).
*   **Cross-Platform Compatibility (Future):** Access your tasks on multiple devices.
